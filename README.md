# Substrace Labs

A premium Next.js website with a stunning Hero section featuring a spotlight effect.

## Getting Started

First, make sure you have Node.js installed (v18 or higher).

Then, install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Spotlight Effect**: Interactive mouse-following spotlight on the hero section
- **Dark Theme**: Premium dark design with grid background
- **Smooth Animations**: Framer Motion animations for a polished feel
- **Responsive Design**: Works beautifully on all screen sizes

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons
